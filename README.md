Library Management System (C Programming)

The Library Management System is a simple C program designed to manage basic library operations such as adding books, displaying records, issuing books, and returning them.
It helps librarians or users keep track of available books and issued ones efficiently using simple file handling and structured programming in C.

How It Works
<br>
The program stores all book details such as Book ID, Title, Author, Quantity, and Status.
Users can add, search, or update book information.
Issuing or returning books updates the availability count.
Data is saved in a file for future access.
